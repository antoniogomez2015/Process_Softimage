

Detalles


tenemos la posibilidad de crear cuerpos blandos "Soft Bodys", los cuales pueden interactura con fuerzas/fields,

y demás interacciones dinámicas entre otros


el workflow, de los sistemas soft bodys, en muy diferente al de los rigid bodys, por que en los soft bodys, no existe

el sistema de "Enviroments", si no, que toda la información se registra de la misma mesh/motive, que hara de soft body,

ej: "Explorer/mesh/Polygon Mesh/SoftBodyOp/", y en esta sección se guardarán todos los elementos 
relacionados con el propio sistema

de soft body, particular de cada mesh/motive, respectivo, como los "Obstacles/ostáculos", "Fuerzas/Fields", otros


podemos crear elementos soft bodys, a partir elemento/objects, completos ó enteros ó a través de selecciones de 

sub-elementos como vertices, otros, para que de esta manera partes de un elemento se mantengan estáticas mientras que

otras tienen propiedades de rigid bodys, dinámicas, el proceso de workflow, para este proceso es el mismo que el proceso

normal para objects, completos

	Obstacle
	________


	
		_Mute:

			_podemos activar ó desactivar las opciones de "Obstacles", asignado al

			sistema soft body, de este cuerpo


		_Names:


			_podemos ver los nombre de los elementos obstacles asígnados al sistema

			soft body, de este cuerpo


		_Obstacles Types:

			_podemos configurar el recubirmiento del tipo de "Obstacle", entre Box.

			Sphere, y Actual Shape


	Obstacles / Physical
	____________________


		_Mute:

			_podemos activar ó desactivar obstáculas en un momento determinado


		_Animatable Deformation:

			_podemos hacer animables las deformaciones dinámicas de los soft bodys


		_Double Face Collision:

			_podemos hacer a los objectos "collider", colisionables por ambas caras

			en de tener múltiples colisiones entre sí



		_Friction:

			_podemos configurar la cantidad de fricción ejercida por los elementos

			"Collider", ó colisionadores sobre los elementos soft bodys


		_Elasticity:

			_podemos configurar la cantidad de elasticidad ejercida por los elementos

			"Collider", ó colisionadores sobre los elementos soft bodys


		_Push Lengh:

			_podemos configurar la cantidad de fuerza de repulsión ejercida por los 

			elementos "Collider", ó colisionadores sobre los elementos soft bodys


		_Tolerance:

			_podemos configurar la cantidad de tolerancia ejercida por los elementos

			"Collider", ó colisionadores sobre los elementos soft bodys
