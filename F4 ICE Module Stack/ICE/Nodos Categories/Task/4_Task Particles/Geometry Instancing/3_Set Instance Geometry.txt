

DETALLES


tenemos la posibilidad de hacer sprites, ó partículas a través, de elementos personalizados como 

instancias de partículas, de esta manera podemos personalizar un sistema de partículas determinado


	Task/Particles/Geometry instancing/Set Instance Geoemtry



////////////////////////////////////////////////////////////////////////////////////////////////////



Set Instance Geoemtry


	_este es el nodo nos permite conectar un geometría ó grupo de geometría de la escena y convertirla 

	en instancias de partículas para el sistema de partículas


	_por lo general este nodo va conectado ha:


		-Set Instance Geoemtry-


			_In:

				_Get Mesh ó group/ Set Instance Geoemtry/ Explorer-Picker


			_Out:

				_"Task/Particles/Emitters/Emit from Geometry/Execute on Emit"

Groups:


	_al crear grupos de elementos "Edit/Create Group", tenemos la posibilidad dentro 

	de las opciones de "Simulate Rigid Bodies", de seleccionar uno de los elementos de

	grupo deseado para la simulación en un momento determinado y animalo si así lo 

	requerimos, para esto:


		_"Simulate Rigid Bodies/Group Object", e indicar a través, de un número entero

		el elemento deseado dentro del group, ó grupo de elementos seleccionado
